# 简介
前后端完全分离的项目。后端共计29个接口，并对所有接口编写测试用例。
包括用户管理、电影分类管理、电影详情管理、电影评论管理的接口。

## 后端技术栈
- 后端：Nodejs+Express
- 数据库：mongoDB
- 自动化：nodemon
- 代码审查：esLint
- 测试框架：Mocha+Chai

## 启动项目
- install dependencies
```sh
$ yarn install
```

- start server 
```sh
$ npm run server
```

- test
```sh
$ npm run test
```
